## PRESENTACIÓN DEL TEMA

El **Perceptrón** es un modelo de red neuronal de **una sola capa**, capaz de resolver únicamente problemas **linealmente separables**.

El **Multi-Layer Perceptron (MLP)** es una red neuronal de **múltiples capas**, que utiliza **funciones de activación no lineales**, permitiendo resolver problemas más complejos.

### Diferencia principal:
- **Perceptrón:** modelo lineal  
- **MLP:** modelo no lineal
